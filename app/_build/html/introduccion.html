Introducción¶

El Sistema de Gestión de Tareas es una herramienta integral diseñada para optimizar y automatizar la gestión de tareas en entornos judiciales y grupos de trabajo. Este sistema ofrece una amplia gama de funcionalidades que permiten una administración eficiente de las tareas, desde su creación hasta su finalización, incluyendo la gestión de alertas, programación y seguimiento.

Módulos principales¶

El sistema está compuesto por los siguientes módulos principales:

  1. Bandeja Principal del Juzgado/Grupo

  2. Vista de tareas personales y de grupo

  3. Alertas de tareas a vencer

  4. Calendario con días inhábiles

  5. Tareas ingresadas desde el Portal del Abogado

  6. Tareas ingresadas desde el Sistema de Expediente del juzgado

  7. Creación de tarea desde el Sistema de Tareas

  8. Notas: creación de notas para una tarea

  9. Labels o etiquetas de tareas

  10. Visibilidad externa tarea

  11. Contenido de multimedia tarea/expediente

  12. Grupos

Cada uno de estos módulos está diseñado para abordar aspectos específicos de la gestión de tareas, proporcionando una solución completa y versátil para las necesidades de los usuarios en el ámbito judicial y otros entornos de trabajo colaborativo.

En las siguientes secciones, se detallará cada uno de estos módulos, sus funcionalidades y cómo interactúan entre sí para crear un sistema de gestión de tareas eficiente y efectivo.
